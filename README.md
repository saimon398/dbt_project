### Тестовый проект для понимания работы DBT

#### Структура проекта
- models:
    - sources: описание источников данных (yml-файл)
    - schemas: описание моделей (yml-файлы)
    - scripts: sql-запросы для сборки моделей


#### Полезные команды

- `dbt parse`
- `dbt ls`
- `dbt run --select <model_name>`
