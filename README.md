### Тестовый проект для понимания работы DBT


В текущем проекте рассматривается такая логическая модель данных (далее ЛМД), как Data Vault. Почему в данном случае был выбран именно DV:

- Гибкое масштабирование хранилища при добавлении новых бизнес сущностей, связей между ними или их описательных атрибутов

- Уменьшение избыточности данных при создании денормализованных аналитических витрин

- Версионирование и поддержка историчности описательных атрибутов
