#### Описание модели
Таблица-сателлит "Пользователь". Хранит атрибуты пользователя (имя, фамилия, email, телефон) из источника MINDBOX. Поддерживает историчность через хэш-разность (`mt_hash_diff`) и временные метки актуальности записи. В отличие от других сателлитов, `mt_valid_from_dttm` берётся напрямую из источника (`valid_from`), а не из момента загрузки.
